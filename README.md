# myfeed-react
A simple RSS reader for my favorite sites.

My feed:
* 4pda.ru
* miped.ru
* reactjs.org/blog
* blog.ubuntu.com
* toster.ru

### How to build
```
$ git clone https://github.com/sezoid/myfeed-react.git
$ cd myfeed-react
$ yarn install
$ yarn build
```

### How to run
```
$ cd myfeed-react
$ yarn start
```

Check your bundles in a 'build' folder.
