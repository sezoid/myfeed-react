# myfeed-react
A simple RSS reader for my favorite sites.

My feed:
* reactjs.org/blog
* overreacted.io
* firebase.googleblog.com
* meduza.io

### How to build
```
$ git clone https://github.com/sezoid/myfeed-react.git
$ cd myfeed-react
$ yarn
$ yarn build
```

### How to run
```
$ cd myfeed-react
$ yarn start
```

Check your bundles in a 'build' folder.
